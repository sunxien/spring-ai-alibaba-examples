# Introduction
This project is used for beginners to develop a new Agent applicaiton with ```spring-ai-alibaba```.
Here are many quick start examples .

# Requirements
1. JDK 21
2. SpringFramework 7.0.0+
3. SpringBoot 4.0+
4. Spring AI OpenAI 2.0.0-M1
5. Spring AI Alibaba 1.1.0.0-RC2

# Spring AI Alibaba Examples
1. Restful API with webflux
2. Multi-Agent (with dynamic building)
